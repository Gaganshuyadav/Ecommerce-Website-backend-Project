
Ecommerce Project 
Project Overview
This is a full-stack ecommerce project built using Node.js, Express.js, MongoDB, and React.js , use stripe to hanble payments. The project features a robust set of functionalities, including user authentication and authorization, payment gateway integration, email notifications, upload multiple images with a single form and with using cloudinary in backend upload multiple images of a single product,and upload images on cloudinary.
